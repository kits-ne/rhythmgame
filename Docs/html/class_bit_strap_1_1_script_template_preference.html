Detailed Description

Draws and syncs a script template.

Script templates are used within the context menu Assets -> Create -> [SomeScript]. Exposing those templates in the editor enables an easy access with the possibility to store different templates as text files and store them in the project. To use script templates go to Edit -> Preferences -> BitStrap.
