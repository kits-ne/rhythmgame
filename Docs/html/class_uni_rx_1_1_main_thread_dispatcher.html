enum UniRx.MainThreadDispatcher.CullingMode
strong
Enumerator
Disabled 

Won't remove any MainThreadDispatchers.

Self 

Checks if there is an existing MainThreadDispatcher on Awake().

If so, the new dispatcher removes itself.

All 

Search for excess MainThreadDispatchers and removes them all on Awake().
